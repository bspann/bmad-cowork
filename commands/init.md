# /bmad-cowork:init

Initialize this project for BMad Method workflows.

## What This Does

1. Creates the `_bmad/` directory with config and status files
2. Creates `_bmad-output/` for planning and implementation artifacts
3. Asks which modules to enable (BMM, TEA, BMB)
4. Sets up workflow tracking

## Instructions

Invoke the `bmad-init` skill to scaffold the project. Follow all steps in the skill.
