# /bmad-cowork:review

Adversarial code review: finds 3-10 issues minimum. APPROVE or REQUEST_CHANGES.

## Usage

`/bmad-cowork:review` — reviews the next story in review status
`/bmad-cowork:review STORY-01-02` — reviews a specific story

## Instructions

Load the `bmad-code-reviewer` agent. Invoke the `bmad-code-review` skill. Follow all steps.
